This directory is reserved for 3D model files (archaeological artifacts, architectural models, sculptures) that will be displayed in interactive viewers.

## Planned Features

- Interactive 3D viewer with rotation, zoom, and pan controls
- Clickable annotations and labels on model surfaces
- Multiple format support (glTF, OBJ, PLY, STL, FBX)
- Material and texture rendering with proper lighting
- Animation playback for animated models
- WebXR support for AR/VR viewing
