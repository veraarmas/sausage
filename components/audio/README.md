This directory is reserved for audio files (oral histories, soundscapes, music, field recordings) that will be embedded in story steps.

## Planned Features

- HTML5 audio player widget embedded in story steps
- Waveform visualization with playback indicator
- Time-coded navigation to specific timestamps
- Transcript integration with synchronized text
- Multiple format support (MP3, WAV, OGG, M4A)
